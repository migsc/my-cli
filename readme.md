# my-cli

## Install

```bash
$ yarn
```

## CLI

```
$ my-cli --help
my-cli

Hello world command

Options:

	--name  Name of the person to greet           [string]
```

## Development

There are 2 available commands:

- `yarn dev` - Start development mode and recompile on change
- `yarn build` - Build a final distributable for npm
